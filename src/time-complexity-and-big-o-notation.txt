Time complexity -: The time taken by algorithm as a export function  of the length of input 

arr = [1, 2, 3, 4]

export function  find(value, arr) {
  for(iterator of arr) {
    if (iterator === value) {
      return true
    }
  }
  return false
}
find(5, arr)

Big-O notation of find export function  is -> O(n)

f(n) = 3n^3 + 5n -> O(n^3)
f(n) = n + 100log(n) -> O(n)
f(n) = 3n^3 + 4n^5 -> O(n^5)
f(n) = 10000 -> O(1)
